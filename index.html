Game of words, for words

Word game!

This is a word game. In a set amount of time YOU have to arrange given letters into a word. Each word will award you with points...
So... You wish to know more about this spectacularly working game with no bugs whatsoever.
>As previously stated you will be given a set amount of letters, which you must arrange.
>Longer words will give you more points.
>You can exchange the points you've already earned for a different word, costing you the amount of points you would've earned for the skipped word. So be careful!
>After the time runs out, you will not be able to make any further changes. And you will be able to save your score in the leaderboard!
>But you mustn't worry! The game can be played MULTIPLE times!
DO have fun......
